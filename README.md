# A Monte Carlo Simulation package for STIX
# Requirements
- ROOT version > 6.0
- cmake
- Geant4.10.05, other geant4 versions may also work but have not been tested
# Compile and run
```sh
cmake .
make
./g4STIX  -m test.mac -o test.root -Ba133
```
